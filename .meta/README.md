# RGVL Platform

- `.github/` — GitHub workflows and issue templates
- `.vscode/` — VS Code workspace settings
- `.devcontainer/` — Dev container configs
- `.docker/` — Docker-related files
- `.envs/`, `.secrets/` — Environment and secret files (not committed)
- `.data/`, `.backups/` — Data and backup storage
- `.test/`, `.testdata/` — Test code and data
- `.scripts/`, `.tools/` — Utility scripts and tools
- `.archive/`, `.legacy/` — Old or archived content

See root README for more info.
